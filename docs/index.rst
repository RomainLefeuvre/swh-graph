.. _swh-graph:

Software Heritage - graph service
=================================

Tooling and service providing fast access to the graph representation of the
Software Heritage archive.  The service is in-memory, based on a compressed
representation of the Software Heritage Merkle DAG (see :ref:`data-model`).
